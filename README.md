# MBH_Team13
Team repository to work on analyses for the OSF project. Team 13 of the The Exploratory ManyBabies Habituation Project.

The Exploratory ManyBabies Habituation Project aims to study visual habituation using data from multiple ManyBabies studies (MB1, MB3 and MB4). Infant looking time (LT) is the main variable of interest across the MB projects.
